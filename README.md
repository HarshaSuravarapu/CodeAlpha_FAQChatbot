FAQChatbot

An AI-powered FAQ Chatbot built using Python, Flask, NLP, and Machine Learning techniques. This chatbot automatically answers frequently asked questions using TF-IDF vectorization and cosine similarity.

🚀 Features
AI-based FAQ response system
NLP-powered text matching
User-friendly web interface
Fast and lightweight
Real-time chatbot responses
Beginner-friendly project
🛠️ Technologies Used
Python
Flask
Pandas
Scikit-learn

⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/your-username/CodeAlpha_FAQChatbot.git
2️⃣ Open Project Folder
cd CodeAlpha_FAQChatbot
3️⃣ Install Required Libraries
pip install -r requirements.txt
▶️ Run the Project
python app.py

Open browser and visit:

http://127.0.0.1:5000
🧠 How It Works
User enters a question
TF-IDF converts text into numerical vectors
Cosine similarity finds the closest matching question
Chatbot returns the most relevant answer
